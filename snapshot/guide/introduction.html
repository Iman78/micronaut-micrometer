1 Introduction

This project integrates Micronaut and Micrometer allowing metrics collection for Micronaut applications.

Release Notes

1.2.x

  • Upgrade to Micrometer '1.2.0'

  • Micronaut 1.1.x minimum version requirement

  • Support for Metrics in GraalVM native image (Thanks to https://github.com/dekstroza)

  • Meter Registry support for SignalFX, Kairos, WaveFront, Azure, Appoptics, Datadog, Influx and Elastic and more (Thanks to Thiago Locatelli)

  • Much of the internals around the composite meter registry and how registries are created have been improved.

1.1.0

  • Upgrade to Micrometer 1.1.2

  • Support for JCache Metrics

  • Support for @Timed annotation

  • New /prometheus Management Endpoint Added. See PrometheusEndpoint

  • Support for AWS CloudWatch. Thanks to Nathan Zender.
